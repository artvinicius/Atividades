# iOS - Capgemini | Projetos

Este repositório é destinado à **entrega de atividades e projetos desenvolvidos durante o curso de iOS da Capgemini**.

Aqui estão reunidos exercícios práticos, projetos e exemplos de código criados ao longo do aprendizado, com o objetivo de praticar lógica de programação, orientação a objetos, conceitos fundamentais avançados da linguagem **Swift**, utilizando o **Xcode** como ambiente de desenvolvimento.

## 📌 Objetivo do Repositório

- Centralizar os projetos do curso
- Demonstrar a evolução no desenvolvimento iOS ao longo do curso

## 🛠 Tecnologias Utilizadas

- Swift
- Xcode
- Programação Orientada a Objetos (POO)

## 📂 Estrutura

Os projetos e atividades estão organizados em pastas conforme os temas e desafios propostos durante o curso.

## 📖 Observações

Este repositório tem fins **educacionais** e é utilizado exclusivamente para acompanhamento e avaliação das atividades do curso iOS - Capgemini.
